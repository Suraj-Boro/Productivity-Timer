Tips

    

    1. Adjust the size of buttons using CTRL + '+' or CTRL + '-' keys to zoom in and out.


    2. Press F11 for fullscreen mode.


    3. For FireFox browser, the layout looks best at  i) 120% zoom (in fullscreen mode)   ii) 100% zoom (in non-fullscreen mode)

                                                 
    4. For Google Chrome browser, the layout looks best at  i)  100% zoom (in fullscreen mode)   ii) 80% zoom (in non-fullscreen mode)
